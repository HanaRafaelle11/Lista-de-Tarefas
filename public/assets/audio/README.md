# Arquivos de Áudio — MyFlowDay

Coloque os arquivos de áudio ambiente nesta pasta. Nomes esperados:

| Arquivo | Descrição |
|---------|-----------|
| `rain.mp3` | Som de chuva |
| `forest.mp3` | Sons da floresta |
| `cafe.mp3` | Ambiente de cafeteria |
| `ocean.mp3` | Ondas do mar |
| `white-noise.mp3` | Ruído branco |

> **Dica**: Você pode obter áudios livres de direitos no [Freesound](https://freesound.org/) ou [Pixabay](https://pixabay.com/sound-effects/).
> Salve-os como `.mp3` com bitrate de 128kbps e duração mínima de 30 segundos (serão colocados em loop automaticamente).
